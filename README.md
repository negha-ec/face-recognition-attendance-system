# face-recognition-attendance-system

A web-based attendance management system using real-time face recognition, built as a BTech mini project.

## Features
- Real-time face recognition using webcam
- Automatic attendance marking
- Web dashboard for managing attendance
- Per-subject attendance tracking
- Manual attendance correction
- Color-coded attendance reports
- Excel export functionality
- Session ID system for each attendance session

## Tech Stack
- **Python** — Core language
- **Flask** — Web framework
- **OpenCV** — Camera and image processing
- **face_recognition** — Face detection and matching
- **SQLite** — Database
- **Pandas** — Excel export
